pub struct AsyncWriterWithChecksum<D, W> { /* private fields */ }
Expand description

Wrapper AsyncWriter implementation that calculates the checksum on the fly. Both Writer and Digest parameters are generic so one can use freely the relevant structures, e.g. BufWriter and Sha256.
